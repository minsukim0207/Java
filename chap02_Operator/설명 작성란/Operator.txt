연산자 (Operator)
연산에 사용하는 기호
간단한 사칙, 비교, 논리, 조건 연산자 등이 있다

1. 산술 연산자 (+, -, *, /, %)
산술 연산자는 숫자 데이터 타입의 변수들을 다룰 때 사용
기본적인 산술 연산을 실행
덧셈, 뺄셈, 곱셈, 나눗셈, 나머지

2. 대입 연산자 (=)
변수에 값을 할당할 때 사용
오른쪽 피 연산자의 값을 왼쪽 피 연잔사에 할당

3. 증감 연산자
변수의 값을 1씩 증가 또는 감소시킬 때 사용
전위 증감 연산자 (++x, --x)
후위 증감 연산자 (x++, x--)

4. 비교 연산자
값을 비교할 때 사용
크기나 등호 비교 등을 수행할 수 있음
두 개의 항 중 어떤 것이 큰지, 작은지, 같은지, 다른지 여부를 검사
비교 연산자의 결과 값은 참(true) 또는 거짓(false)으로 반환
> 왼쪽 항이 크면 참, 아니면 거짓을 반환
< 왼쪽 항이 크면 거짓, 아니면 참을 반환
>= 왼족 항이 크거나 같으면 참, 아니면 거짓을 반환
<= 왼쪽 항이 작거나 같으면 참, 아니면 거짓을 반환
== 두 항의 값이 같으면 참, 아니면 거짓을 반환
!= 두 항의 값이 다르면 참, 아니면 거짓을 반환
