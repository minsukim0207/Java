상속 (Inheritance)
클래스 간의 관계를 나타내며 코드의 재사용성과 확장성을 높이는데 사용
한 클래스가 다른 클래스의 속성과 메서드를 물려받아 사용
기존 클래스의 기능을 기반으로 새로운 클래스를 생성할 수 있음
코드 중복을 줄이고 프로그램을 더 쉽게 유지보수할 수 있다

// 부모 클래스 정의
class 부모클래스 {
	// 부모클래스 정의
}

// 자식 클래스 정의 및 어떤 부모를 상속할 지 결정
class 자식클래스 extends 부모클래스 {
	// 자식클래스 정의
}

접근제어자
부모클래스의 멤버가 자식 클래스에서 접근 가능한지를 제어 가능

메서드 오버라이딩
자식클래서는 부모 클래스의 메서드를 재정의하여 자신의 동작을 정의

super 키워드
자식클래스에서 부모클래스의 멤버를 참조할 때 super 키워드를 사용

다형성
상속을 통해 다형성을 구현할 수 있다
이는 부모 클래스 타입의 변수가 자식 클래스의 객체를 참조할 수 있는 능력
