StringTokenizer 와 Replace
공통점
문자열 처리
*파싱: 데이터를 분해 분석하여 원하는 형태로 조립하고 다시 빼내는 것

차이점
StringTokenizer
문자열을 특정 구분자를 기준으로 토큰으로 분리
입력 문자열을 파싱하거나 특정 형식의 문자열을 분해하는데 유용
CSV파일에서 데이터를 읽거나 특정 형식의 문자열을 파싱하는데 사용

Replace
문자열 내에서 특정 문자열 또는 문자를 다른 문자열 또는 문자로 교체
문자열에서 특정 패턴을 찾아서 변경하는데 사용
ex) 문자열 내의 모든 공백을 다른 문자로 바꿀 때
